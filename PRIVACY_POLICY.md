# Политика конфиденциальности для Global Lens Bot

_Последнее обновление: 29 января 2026г._

Данная Политика конфиденциальности объясняет, какие данные собирает Global Lens Bot (далее — «Бот»), как они используются и защищаются.

---

## 1. Информация, которую мы собираем
Бот может собирать следующую информацию:
- Идентификатор пользователя Discord (User ID);
- Идентификатор сервера (Server ID);
- Изображение профиля пользователя (User avatar) для некоторого функционала Бота;
- Содержимое сообщений, необходимых для выполнения команд Бота, включая личные сообщения, отправленные Боту.

---

## 2. Предоставление разрешений
Используя услуги Бота, пользователь предоставляет согласие на следующие действия:
- Отправку личных сообщений от имени Бота для обеспечения функциональности приложения;
- Использование идентификаторов пользователя (включая Discord ID, никнейм, имя аккаунта и его аватар) исключительно для работы Бота.

---

## 3. Использование данных
Собранные данные применяются только для:
- обеспечения работы функционала Бота;
- ведения журналов модерации;
- диагностики и устранения ошибок (отладки).

Бот не использует данные для рекламы, аналитики или передачи третьим лицам.

---

## 4. Хранение данных
Мы храним только минимальный объём данных и только на период, необходимый для функционирования Бота.  
Журналы и временные данные удаляются автоматически после прекращения использования Бота пользователем или при технической очистке системы.

---

## 5. Удаление данных
Пользователь может запросить удаление данных, связавшись с персоналом проекта Global Lens, имеющий доступ к обновлению актуальной информации.

---

## 6. Изменения
Мы можем обновлять данную Политику без предварительного уведомления. Продолжая использование Бота, вы соглашаетесь с её актуальной версией.
В противном случае, пользователь может отказаться использовать бота, опираясь на пункт 5.
